Component List

Motherboard: Gigabyte B660M DS3H

The Gigabyte B660M DS3H is an excellent choice for its affordable price, PCIe 4.0 support, and solid compatibility with the Intel i5-12400F and RTX 3050, making it a great value motherboard for a performance-oriented build.

Intel Core i5-12400F

The Intel i5-12400F is a good combination for this setup because it offers strong single-core and multi-core performance, making it ideal for gaming and productivity tasks while perfectly complementing the RTX 3050 and other components for balanced, efficient performance.

RTX 3050 8GB

NVIDIA RTX 3050 8GB is a perfect choice for a mid-range setup as it delivers excellent 1080p gaming performance, supporting ray tracing and DLSS, making it an ideal match for the Intel i5-12400F and other components for smooth, high-quality gaming.

16GB-3200MHz Kingston HyperX Fury RAM

The 16GB Kingston HyperX Fury 3200MHz RAM offers excellent performance and reliability for gaming and multitasking, providing a solid foundation for a high-performance PC build.

500GB Kingston NVMe SSD | PCIE 4.0

The 500GB Kingston NVMe SSD PCIe 4.0 provides fast read/write speeds and dependable performance, making it a great option for a mid-range PC build that prioritizes speed, responsiveness, and value.

Corsair CX 550W PSU

The Corsair CX 550W PSU is a great choice for a mid-range PC build with an RTX 3050 8GB and Intel i5-12400F because it offers reliable power delivery, efficiency (80+ Bronze certification), and enough wattage to comfortably support the components without being overkill, ensuring stable performance at a budget-friendly price.
